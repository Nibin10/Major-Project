About

Sonia is an application made available for the blind and visually impaired people. Sonia allows the user to use email service verbally. The user can compose as well as read mail using verbal communication. This application also provides a verbal ebook function, where the user can listen to audio of the respective books selected. Sonia's verbal calculator makes it easy for the user to calculate in Malayalam. Other applications such as a search engine,alert message is also made available by Sonia.

What We Do

E-MAIL

Sonia provides a platform to use the email service verbally. The user can use verbal communication to compose as well as read mail. Messages can be created in a user’s voicemail mailbox and then forwarded to another mailbox. All the instructions are made available in Malayalam. To know more about the service say "Instructions".

Verbal Calculator

Soniaa provides a platform to use the calculator service verbally and in Malayalam. The user can use verbal communication to calculate. Use your voice or the custom keyboard to input calculations and unit conversions easily and quickly. To know more about the service say "Instructions".

Ebook Reader

Ebook platform in Soniaa allows the user to listen to the narration of the books available. These can then be listened to from any electronic device that supports streaming audio, such as a phone, computer, or tablet. To know more about the service say "Instructions".

Search Engine

Soniaa provides a platform to use the search engine in Malayalam. Search engines are able to return results quickly—even with millions of websites online—by scanning the Internet continuously and indexing every page they find. To know more about the service say "Instructions".
